1. conda 잘 들어왔는지 확인 코드
conda --version

2. 혹시 인식되지 않았을 때 붙여 넣을 코드
{
  "python.condaPath": "${env:USERPROFILE}\\miniconda3\\Scripts\\conda.exe",
  "python.terminal.activateEnvironment": true,
  
  "terminal.integrated.profiles.windows": {
    "Command Prompt": {
      "path": "C:\\Windows\\System32\\cmd.exe",
      "args": ["/K", "%USERPROFILE%\\miniconda3\\Scripts\\activate.bat", "2>nul"]
    },
    "PowerShell with Conda": {
      "source": "PowerShell",
      "args": [
        "-NoExit",
        "-NoLogo",
        "-Command",
        "& {& '${env:USERPROFILE}\\miniconda3\\shell\\condabin\\conda-hook.ps1' 2>$null; if ($?) {conda activate base 2>$null}}"
      ]
    }
  },
  "terminal.integrated.defaultProfile.windows": "PowerShell with Conda",
  
  "python.defaultInterpreterPath": "${env:USERPROFILE}\\miniconda3\\python.exe",
  "python.terminal.activateEnvInCurrentTerminal": true
}

2-1. PowerShell이 기본적으로 외부 스크립트 실행을 막고 있어서 생긴 오류
Set-ExecutionPolicy RemoteSigned -Scope CurrentUser

3. conda 환경 생성
3-1. 생성
conda create -n 원하는 환경 이름 python=3.11 -y
3-2. 환경 활성화
conda activate 원하는 환경 이름
3-3. 환경 비활성화
conda deactivate

4. 환경 생성 시 오류 (Terms of Service 오류 해결 - 아래 코드 순서대로 실행)
conda tos accept --override-channels --channel https://repo.anaconda.com/pkgs/main
conda tos accept --override-channels --channel https://repo.anaconda.com/pkgs/r
conda tos accept --override-channels --channel https://repo.anaconda.com/pkgs/msys2

5. 환경 확인
python --version

6. 필요 패키지 설치
6-1. 지오스택 패키지 설치
conda install -c conda-forge madoka geopandas shapely pyproj -y
6-2. pip 도구 최신화화
python -m pip install -U pip setuptools wheel
6-3. 나머지 필요 패키지 설치
pip install -r requirements.txt
6-4. 설치 완료 확인
streamlit --version

7. API 키 파일 생성
ANTHROPIC_API_KEY=your_anthropic_api_key_here

8. app 실행행
streamlit run app.py




etc. 안 잡히면: PATH 수동 등록
Windows 환경 변수에 PATH 수동 등록

a. Windows 키 → "환경 변수" 검색 → "시스템 환경 변수 편집" 클릭
b. 시스템 속성 창 → 하단 "환경 변수(N)" 버튼 클릭
c. 상단 "사용자 변수" 섹션에서
→ "Path" 선택
→ "편집(E)" 버튼 클릭

d. "새로 만들기(N)" 버튼 클릭 후 아래 5개 추가 VS Code 재실행::
C:\Users\<사용자명>\miniconda3
C:\Users\<사용자명>\miniconda3\Scripts
C:\Users\<사용자명>\miniconda3\Library\bin
C:\Users\<사용자명>\miniconda3\Library\usr\bin
C:\Users\<사용자명>\miniconda3\Library\mingw-w64\bin

e. 확인 → 확인 → 확인 (3번)
→ 모든 프로그램 종료
→ Windows 재시작
